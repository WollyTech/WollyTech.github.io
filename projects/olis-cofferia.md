## Overview

**Oli's Cofferia** is a coffee shop management game developed as a school project. Players manage their own coffee shop, serving customers, upgrading equipment, and building a thriving business. The game combines time management mechanics with resource optimization and customer satisfaction systems.

### Key Features

- Time management gameplay
- Customer satisfaction systems
- Shop upgrades and customization
- Recipe creation and experimentation
- Dynamic difficulty progression

### Gameplay Mechanics

Players must balance multiple tasks simultaneously: taking orders, preparing drinks, serving customers, and managing resources. The game rewards efficient multitasking and strategic planning.

### Design Process

This project was developed as part of a game design course, providing an opportunity to explore management game mechanics and player progression systems. The design process included:

- Initial concept development and prototyping
- Iterative playtesting with classmates
- Balancing gameplay systems for optimal difficulty curve
- UI/UX refinement based on user feedback

### Learning Outcomes

Developing Oli's Cofferia taught valuable lessons about:

- Balancing multiple interconnected systems
- Creating intuitive UI for complex interactions
- Designing for progressive difficulty
- Working within time and resource constraints

The project successfully demonstrated the ability to create a complete, polished game experience within an academic setting.
