# Image Placeholders for Wisteria

Place the following images in this directory:

1. **thumbnail.jpg** (Required)
   - Used on project cards
   - Recommended size: 600x400px
   - Should be eye-catching and represent the project

2. **hero.jpg** (Optional banner)
   - Used at the top of the project detail page
   - Recommended size: 1200x400px

3. **screenshot1.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

4. **screenshot2.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

5. **screenshot3.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

6. **screenshot4.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

You can add more screenshots by editing the gallery section in `/projects/wisteria.md`
