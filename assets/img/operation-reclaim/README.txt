# Image Placeholders for Operation Reclaim

Place the following images in this directory:

1. **thumbnail.jpg** (Required)
   - Used on project cards
   - Recommended size: 600x400px
   - Should be eye-catching and represent the project

2. **screenshot1.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

3. **screenshot2.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

4. **screenshot3.jpg** (Gallery image)
   - Recommended size: 800x600px or larger

You can add more screenshots by editing the gallery section in `/projects/operation-reclaim.md`
